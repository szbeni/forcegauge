# Force Gauge with ESP8266

ESP8266 with loadcell connected via HX711 SPI.
Real time plotting via WIFI

# Hardware main parts:
 - ESP8266 MCU
 - 3 buttons
 - OLED screen 0.96" SSD1306
 - Battery with charger
 - USB UART connection
 - HX711 ADC - Load cell
